⌨️ Raccourcis clavier

      
Valables dans n'importe quel terminal du dojo (Apprendre, Bac à sable, Défis, Infiltration…).

      

        
TabAutocomplétion — commandes et chemins de fichiers

        
↑Commande précédente dans l'historique

        
↓Commande suivante dans l'historique

        
Ctrl+RRecherche inversée dans l'historique (façon bash) — tape pour filtrer, Ctrl+R pour le résultat précédent, Entrée pour lancer, Échap pour annuler. (Apprendre, Bac à sable, Exploration)

        
EntréeExécuter la commande

        
ÉchapFermer la fenêtre ouverte (indice, quiz, victoire…)

        
?Afficher / masquer cet écran

      

      
💡 Tape help dans un terminal pour la liste des commandes, ou man <commande> pour le détail d'une seule.
